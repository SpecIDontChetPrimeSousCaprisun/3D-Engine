## Controlles / Controls
### Francais
Z = avancer tout droit
S = reculer
Q = aller a guauche
D = aller a droite
Alt = vérouiller / déverouiller la souris

La caméra est bougée par la souris

### English
Z = forwards
S = backwards
Q = go left
D = go right
Alt = Lock / unlock mouse

The camera is controlled by the mouse
